# サイトマップ・ページ構成

## 情報アーキテクチャ（現行サイト）

```
トップ（top.html）
├── ヘッダー（共通ナビ + CTA: 寄付・入会=join.html / 相談=contact.html）
├── ヒーロー（ミッション）
├── 知りたい情報3カード（体験談/お金と制度/交流・活動）
├── 体験談ダイジェスト（#voice）
├── 年齢別ガイド一覧（#guide：乳幼児/学齢/青年/成人）
├── 私たちについて（ダイジェスト）
├── フッター（団体情報/知りたい情報/年齢別ガイド/ご支援・参加）
└── 固定CTA（join.html / contact.html）

入会案内（join.html）
├── ヘッダー（共通）
├── ヒーロー（入会のご案内）
├── 入会の流れ
├── 会費・サポート内容
├── FAQ
└── CTA（相談・活動導線）

相談・問い合わせ（contact.html）
├── ヘッダー（共通）
├── ヒーロー（まずは相談）
├── 相談方法（電話/フォーム準備中/イベント見学）
├── よくあるご相談
└── CTA（入会・活動導線）

下層共通
├── about / activities / support / voices / guide-*.html は共通ヘッダー/フッター/固定CTA
├── ガイドページは前後ナビ + ガイド一覧への戻り導線
└── フッター項目は「団体情報 / 知りたい情報 / 年齢別ガイド / ご支援・参加」で統一
```

## ページ一覧

### メインページ
- `top.html` - トップページ（アンカー: `#voice` `#guide` `#access`）

### 主要コンテンツページ
- `about.html` - 父の樹会について
- `activities.html` - 活動内容
- `support.html` - お金と制度
- `voices.html` - 体験談

### 年齢別ガイド
- `guide-infant.html` - 乳幼児期ガイド
- `guide-school.html` - 学齢期ガイド
- `guide-youth.html` - 青年期ガイド
- `guide-adult.html` - 成人期ガイド

### CTAページ
- `join.html` - 入会案内
- `contact.html` - 相談・問い合わせ

## ナビゲーション構造

### グローバルナビゲーション
- ホーム (`top.html`)
- 父の樹会について (`about.html`)
- 活動内容 (`activities.html`)
- お金と制度 (`support.html`)
- 体験談 (`voices.html`)
- 年齢別ガイド (`top.html#guide`)

### 固定CTA
- 寄付・入会 (`join.html`) - PCヘッダー右側 / スマホ固定フッター
- 相談・問い合わせ (`contact.html`) - PCヘッダー右側 / スマホ固定フッター

### フッターリンク
- **団体情報**: 父の樹会について、活動内容
- **知りたい情報**: お金と制度、体験談
- **年齢別ガイド**: 乳幼児期、学齢期、青年期、成人期
- **ご支援・参加**: 入会案内、寄付、相談・問い合わせ
