# 父の樹会ホームページ リニューアル案

## 現状分析

### 現在のサイトの特徴
- **組織名**: 障害児者の将来を守る父の樹会
- **設立**: 1978年（千葉大学教育学部付属特別支援学校の父兄が中心）
- **理念**: 「子供の生涯の幸せシステム」作り
- **特徴**: 父親も率先して共同で取り組む点が特色

### 主なコンテンツ
- 父の樹会について
- 活動内容
- スケジュール
- よくあるご質問
- お問合せ
- リンク
- 新着情報（青年学級、生活支援部通信、バザー情報など）

## リニューアル案

### 親しみやすく温かみのあるモダンデザイン 【採用案・詳細版】

**コンセプト**: 「孤立させない家族支援 - 経験と共感でつなぐ」

> [!IMPORTANT]
> **詳細プロンプトによるアップデート**: 
> ターゲットを「母親」中心に再設定し、寄付・会員獲得への導線を強化。
> アクセシビリティ（JIS X 8341-3）への準拠を必須要件とする。

#### 1. プロジェクト概要と目的
- **ミッション**: 障害児者の養育や進路について、父親が率先し共同で取り組み、「子供の生涯の幸せシステム」作りを目指す。
- **デザイン目的**: 会員獲得と寄付金集めを促進する外部向けの情報発信プラットフォームへの転換。
- **ターゲット**: 障害児を育てる母親（父親が主体だが、訪問者は母親が多いことを想定）。
- **ターゲット心理**: 孤独感、将来への不安、多忙感、育児ストレス。
- **喚起する感情**: 安心感、共感、信頼性、温かさ、前向きさ。

#### 2. デザインの方向性（トンマナ）
- **全体イメージ**: 親しみやすく、柔らかい、温かみのあるデザイン。脱・堅苦しさ/医療的な無機質さ。
- **カラースキーム（実装値）**:
  - **メイン**: 淡いグリーン `#81C784`（安心感）
  - **メイン（濃）**: 濃いグリーン `#519657`（文字・強調）
  - **アクセント**: 控えめなオレンジ `#FFB74D`（希望）
  - **サブ**: 暖色系ブルー `#4FC3F7`（信頼・相談）
  - **ベース**: クリーム/ベージュ `#FFFCF5`（温かみ）
  - **テキスト**: 濃いグレー `#2C2C2C`（読みやすさ）
- **フォント**: 
  - `BIZ UDPGothic`（ユニバーサルデザインフォント）
  - `Noto Sans JP`
  - 文字サイズは読みやすさを重視して大きめ（18px基準）に設定。
- **ビジュアル**: 笑顔の子どもや家族、活動風景（ピアサポート、学習会）。スタッフの顔が見える安心感。
- **イラスト**: 手書き風または丸みを帯びた柔らかいタッチ。「仲間ができる」イメージ。

#### 3. 必須コンテンツとレイアウト戦略
- **メインビジュアル**: 「父親の会」の独自性 + 安心感 + ミッションの訴求。
- **コンテンツ優先順位**:
  1. **体験談・ピアサポート**: 「一人じゃない」と感じられる共感コンテンツ。
  2. **制度・費用**: 親御さんが切実に知りたい「生の情報」。
  3. **活動内容**: 楽しさやメリットが伝わる内容。
- **CTA（行動喚起）**: 
  - **PC**: ヘッダー内に「寄付・入会」「相談・問い合わせ」ボタンを配置。
  - **スマホ**: 画面下部に固定フッターとして常時表示。
  - どのページからでもアクセス可能にする。

#### 4. アクセシビリティとユーザビリティ
- **アクセシビリティ（必須）**: 
  - JIS X 8341-3（ウェブアクセシビリティ基盤委員会）を考慮。
  - 文字サイズ、コントラスト比の確保（高齢者・弱視者配慮）。
  - 合理的配慮の提供（2024年4月義務化対応）。
- **ユーザビリティ**:
  - シンプルで美しいデザイン。
  - 「見て分かる」直感的な操作性。
  - 迷わせない明確な階層構造。

#### 情報アーキテクチャ（改訂版）

```
トップページ
├── [CTA固定] 寄付・入会 / 相談・問い合わせ
│
├── メインビジュアル（安心感・ミッション）
│
├── 私たちの想い・ミッション
│
├── 知りたい情報（優先度高）
│   ├── 先輩家族の体験談（共感）
│   ├── お金と制度の話（実用）
│   └── ピアサポート・交流の場（安心）
│
├── 年齢別ガイド
│   ├── 乳幼児期
│   ├── 学齢期
│   └── 青年・成人期
│
├── 父の樹会について
│   ├── スタッフ・メンバー紹介（顔が見える）
│   └── 活動実績
│
└── フッター（サイトマップ・連絡先・寄付導線）
```

#### この案にした理由
1. **組織の理念に合致**: 「子供の生涯の幸せシステム」という温かい理念を、デザインで表現できる
2. **父親の参加促進**: 親しみやすいデザインで、これまで関わりが少なかった父親も参加しやすい
3. **実現性**: 開発難易度が比較的低く、現実的な予算・期間で実現可能
4. **メンテナンス性**: シンプルな構造で、継続的な運用がしやすい
5. **レスポンシブ対応**: スマホからのアクセスも快適で、現代のニーズに対応

#### 実装優先順位
1. トップページとナビゲーション
2. 新着情報セクション（既存のコンテンツを移行）
3. カレンダー機能
4. 各種コンテンツページ
5. お問い合わせフォーム

5. **開発**: 承認後、実装開始
